Function set of text manipulations for all versions of nodejs and major browsers 


## Install

To install from npm:

```sh
$ npm install --save @fav/text
```

Or from github (when npm < 2.7.0 which is not support scoped package):

```sh
$ npm install --save sttk/fav-text
```
